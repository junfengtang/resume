微信小程序:【天气工具类】

2018.02-2018.03

  1. php 编写代码抓取全国省市信息，并且拼接输出 xml 格式。
  2. 天气查询采集中国天气网数据。半小时遍历一遍所有城市，定时缓存天气数据到 Redis。
  3. 使用 springboot 编写业务代码，完成版本1.0，并且购置服务器，配置 SSL 证书，搭建 java 环境将天气 数据接口上线。
  4. 后面进行小程序开发，手绘原型图，编写小程序代码，测试审核上线。
  5. 采用 springcloud 进行业务拆分，拆分为城市信息微服务，天气存储微服务，天气数据微服务。

编写 shell 脚本

  1. 由于平常管理服务器比较多，而且平台杂乱，有阿里云、digitalocean、AWS、腾讯云;每次登陆服务器 要敲长串代码，所以编写 shell 脚本，将服务器分门别类，shell 脚本 alias 到环境变量，方便平时登陆不 同服务器(只要敲几个字母和一个数字就完成登陆哦~)
  2. 利民信息化项目 java 环境的项目部署脚本
  3. 定时拷贝 hybris 日志等等.
